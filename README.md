# 📚 Book Store Management System
## 📌 Overview
The Book Store Management System is a web-based application developed using PHP and MySQL. It allows users to register, log in, and manage books efficiently through CRUD (Create, Read, Update, Delete) operations.
## 🚀 Features

### User Authentication

* User Registration
* User Login
* Secure Session Management
* Logout Functionality

### Book Management

* Add New Books
* View All Books
* Search Books by Title or Author
* Edit Book Details
* Delete Books
## 🛠️ Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* XAMPP
